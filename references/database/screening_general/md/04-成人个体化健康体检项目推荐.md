# 成人个体化健康体检项目推荐专家共识（2023版）

> 《健康体检与管理》杂志编委会 中国医师协会医师健康管理与医师健康保险专业委员会
> 北京医学会健康管理学分会 北京市体检质量控制和改进中心
> 发表于《健康体检与管理》2023年7月第4卷第3期

---

## 一、共识概述

个体化健康体检是医务人员根据受检者健康风险评估和疾病筛查意愿，基于个人基本信息、家族病史、既往疾病状况及常见慢性病风险评估结果，制定的个体化健康体检项目组合。

**核心原则**: 通过高质量检前问卷评估，针对性制定个体化体检项目，实现最大健康获益的同时兼顾最佳成本效益。

### 个体化体检实施路径
```
受检者 → 检前问卷评估（基础问卷+慢性病风险+躯体活动能力）
  → 医务人员依据测评结果 + 专病筛查指南制定个体化方案
    → 基本项目（必选）+ 个体化加项（按风险分层）
      → 检后健康管理干预
```

## 二、个人健康信息采集（检前评估）

### 基础问卷（5个维度）
| 维度 | 内容 |
|------|------|
| **人口学特征** | 年龄、性别、婚育史、受教育程度、职业、居住地 |
| **个人史** | 家族史、既往病史（含患病年限、服药年数）、手术史 |
| **躯体症状** | 头晕、疼痛、胸闷、咳嗽、恶心、腹胀、出血、大小便异常等 |
| **生活方式** | 饮食、吸烟（年限+量）、饮酒（年限+量）、运动（频率+类型） |
| **睡眠状况** | 睡眠时长、质量、影响因素 |

### 慢性病风险筛查问卷
- 心脑血管疾病风险筛查问卷
- 恶性肿瘤风险筛查问卷
- 慢性阻塞性肺病风险筛查问卷
- 代谢相关疾病风险筛查问卷

### 躯体活动能力筛查问卷
- PAR-Q问卷（体力活动准备问卷）
- 疾病史和运动习惯信息采集

## 三、健康体检基本项目（所有成年人必选）

| 大类 | 具体项目 |
|------|----------|
| **体格检查** | 一般检查（身高/体重/腰围/BMI/血压）、物理检查（内科/外科/眼科/耳鼻喉/口腔/妇科） |
| **实验室检查** | 血常规、尿常规、便常规、生化全项（肝肾功能/血脂/血糖/尿酸）、免疫学检查、肿瘤血清学检查、细胞学检查 |
| **辅助检查** | 心电图、X线检查（胸片）、超声检查（腹部+盆腔）、其他影像学检查 |

## 四、个体化体检加项推荐（按疾病风险）

### 4.1 心脑血管疾病

#### 高血压风险人群
| 项目类型 | 推荐项目 |
|----------|----------|
| **首选** | 血浆同型半胱氨酸、hs-CRP、电解质（Na/K/Cl）、肌酐、尿微量白蛋白、超声心动图、颈动脉超声 |
| **备选** | 肾高血压4项、24小时动态血压、踝臂血压指数、脉搏波传导速度、冠脉钙化积分、肾动脉超声、眼底检查 |
| **深度** | 呼吸睡眠监测、头颅MRA、经颅多普勒超声、人体成分分析、血管内皮功能检查、肾上腺CT |

#### 冠心病风险人群
| 项目类型 | 推荐项目 |
|----------|----------|
| **首选** | NT-proBNP/BNP、高敏肌钙蛋白、载脂蛋白、脂蛋白a、心电图、超声心动图、颈动脉超声 |
| **备选** | 24小时动态心电图、运动平板试验、踝臂血压指数 |
| **深度** | 冠脉CTA、负荷心肌灌注显像、心肺运动试验、无创冠脉血流储备分数 |

#### 脑卒中风险人群
| 项目类型 | 推荐项目 |
|----------|----------|
| **首选** | 神经内科查体、脑血管功能积分、hs-CRP、血脂全项（含脂蛋白a）、凝血功能、同型半胱氨酸、颈动脉超声、超声心动图 |
| **备选** | 甲状腺功能、抗磷脂抗体3项（狼疮抗凝物/β2-GP1/抗心磷脂抗体）、糖化血红蛋白、头部MRI+MRA |
| **深度** | 24小时动态心电图、24小时动态血压、冠脉CTA |

### 4.2 恶性肿瘤筛查（按癌种）

| 癌种 | 首选筛查手段 | 辅助/备选手段 |
|------|-------------|--------------|
| **肺癌** | 低剂量螺旋CT（LDCT） | 肺癌自身抗体、SCC/NSE/CYFRA21-1/pro-GRP/CEA、SHOX2/RASSF1A/PTGER4基因甲基化 |
| **结直肠癌** | 结肠镜（金标准）→ 免疫化学法粪便隐血（序贯首选） | 血浆Septin9/BCAT1甲基化、粪便SDC2基因检测、多靶点粪便FIT-DNA、CEA |
| **乳腺癌** | 乳腺超声 + 乳腺X线联合（中国女性首选） | 乳腺MRI（BRCA突变高危人群）、A-BUS、BRCA1/2基因检测 |
| **胃食管癌** | 上消化道内镜（金标准）→ 碳13/14呼气试验+胃蛋白酶原（序贯首选） | 胃泌素17、HP抗体/分型、CEA/CA19-9/CA72-4/CA24-2、磁控胶囊胃镜 |
| **肝癌** | 超声 + AFP（首选联合） | AFP-L3%+DCP（PIVKA-II）三联、肝炎病毒检测、肝脏瞬时弹性成像、CTC |
| **宫颈癌** | 宫颈细胞学（巴氏/液基）或 HPV核酸检测（单独均可首选） | HPV E6/E7 mRNA、SCC |
| **前列腺癌** | PSA（总+游离）、前列腺超声 | 前列腺MRI |

### 4.3 慢性阻塞性肺病（COPD）
- 肺功能检查（首选）
- 胸部CT、二维超声心动图
- 心肺运动试验、过敏原检测、IgE

### 4.4 代谢相关疾病

#### 糖尿病
- 空腹胰岛素+C肽、餐后2小时血糖+胰岛素+C肽
- 糖化血红蛋白、糖化白蛋白
- 尿微量白蛋白肌酐比、1型糖尿病自身抗体谱（GADA/ICA/IA-2A/IAA）
- 眼底照相、颈动脉超声、下肢动脉超声

#### 骨质疏松症
- 床式双能X线骨密度（DXA，腰椎+股骨近端+桡骨远端1/3）— 诊断金标准
- 前臂DXA/跟骨定量超声 — 筛查
- 血钙/磷、25羟维生素D、碱性磷酸酶、骨代谢指标（PINP/BALP/CTX/OC）
- 甲状旁腺激素、性激素6项（绝经前女性）

### 4.5 认知功能障碍
- 量表（MMSE、MoCA、AD8）+ 神经系统检查
- 电解质、血钙、APOEε4等位基因检测
- 血浆Aβ蛋白/tau蛋白
- 头部MRI（含海马冠状位）± PET-MRI/PET-CT

## 五、心理健康筛查
- **主观评价**: 量表问卷（焦虑/抑郁/睡眠量表）
- **客观评价**: 心率变异性（HRV）分析、自主神经功能评估

## 六、躯体活动能力筛查
- 身体姿态评估（静态+动态）
- 运动素质评估（反应时/握力/背力/纵跳/闭眼单脚站/俯卧撑/坐位体前屈等）
- 心肺耐力评估（6分钟步行/心电图运动试验/心肺运动试验）
- 人体成分分析、骨密度、静息代谢分析

## 七、中医治未病筛查
中医检测方法作为传统体检项目的补充，通过中医体质辨识、经络检测等手段对健康状况进行全方位评估。
