# 胃癌筛查指南

> 来源：赫捷 等. 中国胃癌筛查与早诊早治指南（2022，北京）
> 国家癌症中心
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **发病第三位**：2022年新发胃癌约36-39万例
- **中标发病率**：28.7/10万；男性 39.0/10万，女性 17.8/10万
- **高发地区**：中国胃癌发病率存在明显地区差异，西北和东部沿海地区较高
- **特点**：中国胃癌以**肠型胃癌**为主（Correa级联：正常胃黏膜→慢性胃炎→萎缩性胃炎→肠化生→异型增生→胃癌）

> 肠型胃癌遵循Correa级联反应模式，提供了**长达10年+** 的筛查干预窗口

## 高危对象

### 具备以下任一因素即为高危：

1. **年龄>=40岁**，有腹痛、腹胀、反酸、烧心等上腹部不适症状
2. 有慢性胃炎、**胃黏膜萎缩/肠上皮化生**、胃息肉、残胃、慢性胃溃疡等癌前病变
3. 幽门螺杆菌（**Hp**）感染者（Hp是I类致癌物）
4. **高盐饮食**、腌制食品（含亚硝酸盐）长期摄入者
5. 重度吸烟（>=20年包）、重度饮酒者
6. 有胃癌家族史（一级亲属）
7. 有Li-Fraumeni综合征、遗传性非息肉病性结肠癌（HNPCC/Lynch综合征）家族史

## 筛查方法

### 胃镜检查

**胃镜+病理活检** 是胃癌筛查的金标准。

| 性能指标 | 数据 |
|----------|------|
| 灵敏度 | **92.10%** |
| 特异度 | **96.20%** |
| 优势 | 可直接观察+取活检+切除早癌病变 |

### 其他筛查方法

| 方法 | 说明 |
|------|------|
| **血清胃蛋白酶原（PG）** | PG I/II比值降低提示萎缩性胃炎 |
| **Hp抗体/呼气试验** | 检测幽门螺杆菌感染 |
| **血清胃泌素-17（G-17）** | 辅助评估胃黏膜状态 |
| **上消化道钡餐** | 已基本被胃镜取代 |

### 新型胃癌风险评分系统
结合年龄、性别、PG I/II比值、G-17、Hp抗体5项指标，可将人群分为低危、中危、高危三层，指导筛查频率。

## 筛查建议

### 基于风险分层的筛查

| 风险等级 | 筛查方案 |
|----------|----------|
| **低危**（无危险因素） | 45岁起，每3-5年1次胃镜或风险评分 |
| **中危**（Hp阳性/轻度萎缩） | 根除Hp后，每1-3年1次胃镜 |
| **高危**（萎缩/肠化/家族史） | **每年1次**胃镜+病理检查 |
| **极高危**（重度异型增生） | 立即内镜下治疗 |

### Hp根除
- Hp阳性者建议**根除治疗**
- 根除Hp可降低胃癌风险约 **35-50%**
- 已有萎缩/肠化生者，根除Hp后仍需定期随访

## 筛查流程

```
40岁以上人群
    → 胃癌风险评分（PG + G-17 + Hp抗体）
    → 低危 → 每3-5年复查
    → 中危 → Hp根除 + 每1-3年胃镜
    → 高危 → 每年1次胃镜
              → 正常 → 继续每年1次
              → 异型增生 → 内镜下治疗
```

## 癌前病变管理

| 病变等级 | 随访频率 |
|----------|----------|
| 慢性萎缩性胃炎（轻度） | 每2-3年1次胃镜 |
| 慢性萎缩性胃炎（重度） | 每1-2年1次胃镜 |
| 肠上皮化生 | 每1-2年1次胃镜 |
| 低级别上皮内瘤变 | 每6-12个月1次胃镜 |
| 高级别上皮内瘤变 | **立即内镜下切除** |

## 预防建议

1. **根除幽门螺杆菌（Hp）感染**（最重要的一级预防措施）
2. 健康饮食习惯：**不暴饮暴食**，定时定量
3. 减少熏制、腌制、高盐食物摄入；增加新鲜蔬菜水果
4. 戒烟、少喝或不喝烈性酒
5. 放松心情，合理减压
6. 高危人群坚持定期胃镜筛查
