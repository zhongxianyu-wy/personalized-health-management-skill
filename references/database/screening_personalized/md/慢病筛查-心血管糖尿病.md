## 高血压（hypertension）
- 分类：cardiovascular ｜ 来源：心血管疾病筛查与管理指南.md

**诊断标准：**
- 收缩压 ≥ 140 mmHg
- 舒张压 ≥ 90 mmHg OR

**分级：**
| 分级 | 描述/范围 | 管理 |
|---|---|---|
| 正常血压 | <120 / DBP <80 | 保持健康生活方式 |
| 正常高值 | 120-139 / DBP 80-89 | 生活方式干预 |
| 高血压1级（轻度） | 140-159 / DBP 90-99 | 生活方式干预+评估药物治疗 |
| 高血压2级（中度） | 160-179 / DBP 100-109 | 药物治疗+生活方式干预 |
| 高血压3级（重度） | ≥180 / DBP ≥110 | 立即药物治疗+专科就诊 |

**筛查推荐：**
- 一般成年人（≥18岁）：血压测量（每年至少1次）
- 高危人群（≥40岁或高危因素）：血压测量（每3-6个月1次）
- 确诊高血压患者：动态血压监测（按医嘱）

**⚠ 数据缺口：**
- 原文未提供颈动脉超声/心脏超声/冠脉CT的具体筛查频率

## 2型糖尿病（diabetes_mellitus_type2）
- 分类：metabolic ｜ 来源：糖尿病防治指南.md

**诊断标准：**
- 空腹血糖 ≥ 7.0 mmol/L
- OGTT 2h血糖 ≥ 11.1 mmol/L
- 糖化血红蛋白 ≥ 6.5 %
- 随机血糖 ≥ 11.1 mmol/L (需伴典型症状)

**治疗目标（一般成人）：** HbA1c <7.0%；FPG 4.4-7.0 mmol/L；餐后血糖 <10.0 mmol/L；血压 <130/80 mmHg；LDL-C <2.6 mmol/L
**治疗目标（合并CVD）：** LDL-C <1.8 mmol/L

**并发症筛查：**
- 糖尿病肾病（DKD）：UACR + eGFR，每年1次
- 糖尿病视网膜病变（retinopathy）：原文未明确，原文未明确
- 糖尿病神经病变（neuropathy）：原文未明确，原文未明确

**筛查推荐：**
- 高危人群（≥40岁/BMI≥24/家族史/高血压/血脂异常）：空腹血糖 + OGTT（每年1次）；HbA1c（辅助筛查）

**⚠ 数据缺口：**
