# 食管癌筛查指南

> 来源：赫捷 等. 中国食管癌筛查与早诊早治指南（2022，北京）
> 国家癌症中心
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **死亡第五位**：2022年死亡18.75万例
- **中标发病率**：15.0/10万（农村 11.0/10万 > 城市 6.2/10万）
- **高发地区**：太行山脉周边（河南、河北、山西）、苏北、四川盆地、闽粤沿海
- **极高发区**：某些地区发病率可达 **50/10万** 以上
- **病理类型**：中国以**鳞状细胞癌**为主（>90%），与西方国家以腺癌为主不同

> 食管癌是典型的地域分布不均癌症，特定高发区应作为筛查重点

## 高危对象

1. 来自食管癌**高发地区**（太行山区、苏北、川北、闽南等）
2. 有上消化道症状（**吞咽困难**、胸骨后疼痛/烧灼感、食物滞留感等）
3. 有食管癌**家族史**（一级亲属，风险增加2-5倍）
4. 有食管癌前病变（**Barrett食管**、食管上皮内瘤变、贲门失弛缓症）
5. 具有食管癌高危因素：
   - **吸烟 + 重度饮酒**（协同作用，风险倍增）
   - **喜食烫食/热饮**（>=65°C，IARC 2A类致癌物）
   - 超重/肥胖（腺癌风险）
   - 头颈部或呼吸道鳞癌史
6. 患有**胃食管反流病（GERD）**（增加Barrett食管和腺癌风险）
7. 感染人乳头瘤病毒（HPV）（鳞癌相关）

## 筛查方法

### 内镜检查（金标准）

| 技术 | 灵敏度 | 特异度 | 说明 |
|------|--------|--------|------|
| 白光内镜 | ~46% | ~85% | 基础方法，灵敏度较低 |
| **碘染色内镜** | **100%** | **97%** | 鳞癌筛查首选！碘不染区取活检 |
| 窄带成像（NBI） | 较高 | 较高 | 辅助判断病变边界 |
| 放大内镜 | 较高 | 较高 | 观察微血管和微结构 |

> **碘染色内镜（卢戈氏碘液）** 是中国食管鳞癌筛查的推荐方法

### 其他筛查方法
- **食管脱落细胞学**（拉网/气囊）：传统方法，灵敏度不高，已逐渐被内镜取代
- **血清肿瘤标志物**（SCC、CEA、CYFRA21-1）：辅助诊断，不用于初筛

## 筛查建议

### 筛查起始年龄
- 高发区人群：**40岁** 起
- 有家族史者：40岁或比最小发病年龄小10岁

### 筛查频率（基于病理结果）

| 病理结果 | 筛查频率 | 处理 |
|----------|----------|------|
| 正常鳞状上皮 | 每 **2年** 1次内镜 | 维持筛查 |
| 食管炎/轻度炎症 | 每 **2年** 1次 | 对症治疗 |
| **轻度异型增生** | 每 **1年** 1次内镜 | 密切随访 |
| **中度异型增生** | 每 **半年** 1次内镜 | 考虑内镜下治疗 |
| **重度异型增生/原位癌** | — | **立即内镜下治疗（ESD/EMR）** |

## 筛查流程

```
40岁以上高发区或高危人群
    → 碘染色内镜检查
    → 正常/炎症 → 2年后复查
    → 轻度异型增生 → 1年后复查
    → 中度异型增生 → 半年后复查
    → 重度异型增生/早期癌 → 内镜下治疗（ESD）
```

## 预防建议

1. **不吸烟或戒烟**
2. 少量饮酒或不饮酒（**吸烟+饮酒是食管癌最强的可改变危险因素**）
3. 合理饮食，多食用新鲜水果蔬菜（富含维生素C、E和叶酸）
4. 增强运动，保持健康体重
5. **不食用烫食、不饮用烫水**（食物温度<65°C）
6. 减少腌制食品（含亚硝胺）摄入

## 新兴筛查技术

食管癌目前**缺乏标准无创筛查方法**，传统碘染色内镜具有侵入性且覆盖有限。

> **MCED血液检测**：[[../吉早安/癌种覆盖|吉早安]] 对食管癌的敏感性为 **79.2%**（Stage I: 54.5%），为食管癌无创筛查提供了新选择。
