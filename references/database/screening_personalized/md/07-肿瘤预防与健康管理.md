# 肿瘤预防与健康管理指南（2025版综合）

> 综合 上海市抗癌协会《居民常见恶性肿瘤筛查和预防推荐（2025版）》 + 《健康体检机构防癌体检指南（2025年版）》

---

## 一、肿瘤三级预防体系

| 预防级别 | 目标 | 措施 |
|----------|------|------|
| **一级预防** | 减少癌症发生（病因预防） | 控烟限酒、疫苗接种、健康生活方式、环境致癌物规避 |
| **二级预防** | 早期发现、早期诊断、早期治疗 | 癌症筛查（影像/内镜/肿瘤标志物/液体活检）、防癌体检 |
| **三级预防** | 改善预后、提高生存质量 | 规范化治疗、康复管理、定期随访、心理支持 |

---

## 二、一级预防：可干预的癌症风险因素

### 明确致癌风险
| 风险因素 | 相关癌种 | 预防建议 |
|----------|----------|----------|
| **吸烟** | 肺癌、口腔癌、喉癌、食管癌、膀胱癌、胰腺癌等15+癌种 | 戒烟（所有形式烟草），避免二手烟 |
| **饮酒** | 口腔癌、食管癌、肝癌、结直肠癌、乳腺癌 | 限酒（男性≤25g/d，女性≤15g/d），最好不饮酒 |
| **超重/肥胖** | 食管腺癌、结直肠癌、乳腺癌（绝经后）、子宫内膜癌、肾癌、肝癌 | BMI维持在18.5-24，腰围男<90cm/女<85cm |
| **不健康饮食** | 结直肠癌（加工肉/红肉）、胃癌（腌制食品） | 增加蔬果/全谷物/膳食纤维，减少加工肉和红肉 |
| **缺乏运动** | 结直肠癌、乳腺癌、子宫内膜癌 | ≥150分钟/周中等强度有氧运动 |
| **职业暴露** | 石棉→间皮瘤/肺癌、苯→白血病 | 职业防护、定期健康监护 |

### 感染性因素及疫苗
| 病原体 | 相关癌种 | 预防措施 |
|--------|----------|----------|
| **HPV**（人乳头瘤病毒） | 宫颈癌、肛门癌、口咽癌、阴茎癌 | **HPV疫苗**（9-45岁，最佳9-14岁接种） |
| **HBV/HCV**（乙肝/丙肝） | 肝细胞癌 | HBV疫苗接种、抗病毒治疗、避免血液暴露 |
| **幽门螺旋杆菌（HP）** | 胃癌、胃MALT淋巴瘤 | 检测+根除治疗（四联疗法） |
| **EBV** | 鼻咽癌、Burkitt淋巴瘤 | 无特异性疫苗，提升免疫力 |
| **HIV** | 卡波西肉瘤、非霍奇金淋巴瘤 | 抗病毒治疗、免疫监测 |

## 三、二级预防：各癌种筛查推荐

### 肺癌
| 项目 | 推荐 |
|------|------|
| **目标人群** | 50-80岁，吸烟≥20包年，或戒烟<15年 |
| **首选方案** | 低剂量螺旋CT（LDCT）每年一次 |
| **辅助** | 肺癌自身抗体检测、外周血甲基化检测 |
| **不推荐** | 胸部X线作为肺癌筛查手段 |

### 乳腺癌
| 项目 | 推荐 |
|------|------|
| **一般风险（40-49岁）** | 乳腺超声 ± 乳腺X线（1-2年/次） |
| **一般风险（50-69岁）** | 乳腺X线 ± 超声（1-2年/次） |
| **高危人群** | 乳腺MRI每年一次 + X线/超声；BRCA1/2突变携带者考虑预防性干预 |
| **高危标准** | 明确遗传突变（BRCA1/2/TP53/PTEN等）、胸部放疗史、≥2位一级亲属患乳腺癌 |

### 结直肠癌
| 项目 | 推荐 |
|------|------|
| **目标人群** | 45-75岁（一般风险） |
| **首选方案** | 结肠镜（每10年） |
| **序贯筛查首选** | 免疫化学法粪便隐血（FIT，每年）→ 阳性者结肠镜 |
| **备选** | 多靶点粪便FIT-DNA（每3年）、血浆SEPT9/SDC2甲基化检测 |
| **高危人群** | 结直肠癌家族史（一级亲属<60岁发病）、Lynch综合征、FAP、炎症性肠病 |

### 胃癌
| 项目 | 推荐 |
|------|------|
| **目标人群** | 40岁以上，有以下任一：HP感染、胃癌家族史、慢性萎缩性胃炎、胃溃疡、吸烟、高盐腌制饮食 |
| **首选方案** | 上消化道内镜（每1-3年） |
| **序贯筛查** | 碳13/14呼气试验+胃蛋白酶原+胃泌素17 → 分层后内镜 |
| **一级预防** | HP根除（降低胃癌风险30-40%） |

### 肝癌
| 项目 | 推荐 |
|------|------|
| **目标人群** | 肝硬化（任何病因）、慢性HBV/HCV感染、肝癌家族史 |
| **首选方案** | 腹部超声 + AFP（每6个月） |
| **增强方案** | AFP-L3% + DCP（PIVKA-II）联合检测（提高早期检出率） |
| **不推荐** | 普通CT作为肝癌常规筛查 |

### 食管癌
| 项目 | 推荐 |
|------|------|
| **目标人群** | 40岁以上+吸烟/饮酒/上消化道癌家族史/Barrett食管/贲门失弛缓症 |
| **首选方案** | 上消化道内镜（每2-3年）+ 卢戈碘染色（鳞癌高危区） |

### 宫颈癌
| 项目 | 推荐 |
|------|------|
| **目标人群** | 25-65岁女性 |
| **25-29岁** | 宫颈细胞学（TCT，每3年） |
| **30-65岁** | 宫颈细胞学+HPV联合检测（每5年）或 单独HPV核酸检测（每5年） |
| **一级预防** | HPV疫苗（9-45岁接种，越早越好） |

### 前列腺癌
| 项目 | 推荐 |
|------|------|
| **目标人群** | 50岁以上（有家族史者45岁起） |
| **方案** | PSA检测 + 直肠指检/经直肠超声 |
| **注意** | 充分知情同意（PSA筛查存在过度诊断争议）；PSA轻度增高可先复查观察 |

### 胰腺癌
| 目标人群 | 筛查方案 |
|----------|----------|
| 遗传性胰腺癌高危（BRCA2/PALB2/STK11/CDKN2A突变、≥2位一级亲属胰腺癌） | EUS或MRCP每年一次 |

### 卵巢癌
| 目标人群 | 筛查方案 |
|----------|----------|
| BRCA1/2突变携带者（35-40岁以上） | CA125 + 经阴道超声每6-12个月（有限获益证据） |
| 一般风险人群 | **不推荐常规筛查**（缺乏降低死亡率的证据） |

## 四、防癌体检机构实施要点

### 4.1 体检前评估
- **结构化问卷**: 采集家族史（三级亲属）、疾病史、症状、生活方式
- **风险评估分层**: 基于问卷+既往检查结果→低/中/高危分层
- **知情同意**: 向受检者说明各筛查方法的获益、风险和局限性

### 4.2 体检方案制定原则
```
一般风险人群 → 基本项目（体格检查+实验室+超声+胸片）
中危人群 → 基本项目 + 针对性加项（如高危癌种的影像/内镜/肿瘤标志物）
高危人群 → 基本项目 + 全程防癌筛查套餐 + 遗传咨询（如适应）
```

### 4.3 体检后管理
| 结果 | 处理 |
|------|------|
| **阴性** | 按推荐间期定期复查 |
| **临界/不确定** | 短期复查（3-6个月）或进一步影像学检查 |
| **阳性（可疑）** | 转诊至相应专科明确诊断 |
| **确诊癌症** | 转肿瘤科规范化治疗 |

### 4.4 质量控制
- 筛查技术应遵循国家标准和行业规范
- 体检机构应建立筛查阳性病例的**跟踪随访机制**
- 鼓励参与国家/地区癌症筛查登记系统

## 五、健康生活方式推荐

| 类别 | 推荐 | 证据强度 |
|------|------|----------|
| **戒烟** | 不吸烟、避免所有烟草制品 | I类证据 |
| **限酒** | 最好不饮酒 | I类证据 |
| **体重管理** | BMI 18.5-24 | I类证据 |
| **运动** | ≥150分钟中等强度/周 或 ≥75分钟高强度/周 | I类证据 |
| **饮食** | 多蔬果/全谷物/膳食纤维；红肉<500g/周；加工肉尽量少吃 | II类证据 |
| **母乳喂养** | 有助于降低母亲乳腺癌风险 | II类证据 |
| **防晒** | 避免过度日晒，使用防晒霜（防皮肤癌/黑色素瘤） | II类证据 |
