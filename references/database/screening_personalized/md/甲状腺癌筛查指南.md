# 甲状腺癌筛查指南

> 来源：中华医学会健康管理学分会 体检人群甲状腺结节风险评估与健康管理专家共识
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **发病第三位**：2022年新发46.61万例（但死亡率极低）
- **趋势**：中国甲状腺癌发病率**急剧上升**，但死亡率保持稳定（过度诊断效应）
- **结节患病率**：体检人群中超声检出甲状腺结节比例高达 **20-76%**
- **恶性比例**：甲状腺结节中仅约 **5-10%** 为恶性

> 甲状腺结节在体检人群中非常常见，多数属于良性病变，关键在于**精确风险评估，避免过度诊疗**

## 高危对象

### 临床高危因素

1. 童年期**头颈部放射线暴露**史（最重要危险因素）
2. 全身放射治疗史（骨髓移植前全身照射等）
3. 有甲状腺癌家族史（尤其一级亲属）
4. 有甲状腺癌相关遗传综合征：
   - **MEN2型**（多发性内分泌腺瘤病2型）→ 髓样癌
   - 家族性甲状腺髓样癌（FMTC）
   - Cowden综合征、Gardner综合征等
5. 甲状腺结节**快速增长**或出现压迫症状（声音嘶哑、吞咽困难、呼吸困难）

## 筛查方法

### 甲状腺超声（首选筛查）

| 参数 | 说明 |
|------|------|
| 方法 | 高频超声（>=7.5MHz探头） |
| 优势 | 无创、无辐射、可重复、灵敏度和分辨率高 |
| 内容 | 结节大小、形态、边界、回声、钙化、血流、颈部淋巴结 |

### 超声恶性征象（TI-RADS评估依据）

| 征象 | 恶性风险 |
|------|----------|
| 实性低回声 | + |
| 微小钙化（<1mm） | ++ |
| 边界不规则/分叶 | ++ |
| 纵横比>1（垂直生长） | +++ |
| 甲状腺外侵犯 | +++ |
| 颈部淋巴结异常 | ++++ |

### 细针穿刺活检（FNA）
- TI-RADS >=4类 → 建议FNA
- Bethesda细胞学报告系统（I-VI级）

## 筛查建议

### "筛查-评估-分层-干预-随访-康复"全链条模式

#### TI-RADS 分级管理

| TI-RADS | 恶性风险 | 管理建议 | 随访 |
|---------|----------|----------|------|
| **1类** | 0% | 正常甲状腺，无需特殊处理 | 常规体检 |
| **2类** | 0% | 良性囊性/海绵状结节 | 常规体检 |
| **3类** | <5% | 低度可疑 | 1-2年超声复查 |
| **4A类** | 5-10% | 中度可疑 | FNA，或6个月超声复查 |
| **4B类** | 10-50% | 中高度可疑 | **FNA** |
| **4C类** | 50-85% | 高度可疑 | **FNA** |
| **5类** | >85% | 极高度可疑 | **FNA / 手术** |
| **6类** | 100% | 已证实恶性 | 手术治疗 |

#### FNA适应证
- 结节**最大径>=1cm** + 超声中高度可疑征象
- 结节**<1cm**但有颈部淋巴结异常/甲状腺外侵犯/童年放射史/家族史
- 结节短期内显著增大（>=20%且>=2mm在两个维度上）

## 预防建议

1. 避免不必要的颈部放射线暴露（尤其儿童）
2. 保持健康生活方式
3. 体检中常规包含甲状腺超声
4. 确诊结节后遵循**规范随访路径**，避免不必要的手术

> 大多数甲状腺癌（乳头状微小癌）进展缓慢，**积极监测**也是可选策略
