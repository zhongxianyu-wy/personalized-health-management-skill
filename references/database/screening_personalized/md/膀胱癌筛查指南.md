# 膀胱癌筛查指南

> 来源：膀胱癌早诊早治专家共识（2024年版）
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **特点**：中国膀胱癌发病率和死亡率相对较低，但城市地区呈上升趋势
- **主要病理类型**：尿路上皮癌（>90%）
- **性别差异**：男性发病率约为女性的3-4倍

## 高危对象

1. **长期吸烟者**（吸烟是膀胱癌**最重要**的危险因素，归因风险约50%）
2. 职业暴露史（**芳香胺、苯胺染料、橡胶、油漆、皮革、印刷**等化学物质接触者）
3. 慢性膀胱炎、膀胱结石、长期留置导尿管、血吸虫感染
4. 盆腔**放疗史**（宫颈癌/直肠癌放疗后）
5. 有膀胱癌家族史
6. 长期使用含**马兜铃酸**的中草药者
7. 饮用水中**砷含量高**地区的居民

## 筛查方法

### 无创筛查

| 方法 | 灵敏度 | 特异度 | 说明 |
|------|--------|--------|------|
| **尿常规** | — | — | 基础筛查，检测镜下血尿 |
| **尿细胞学** | 13%-75% | 87%-95% | 对高级别病变灵敏度高，低级别病变低 |
| **尿NMP22** | 63%-98% | 94%-99% | FDA批准的膀胱癌标志物 |
| **尿液FISH**（UroVysion） | 较高 | 较高 | 染色体异常检测 |

### 确诊方法

| 方法 | 说明 |
|------|------|
| **膀胱超声** | 无创初筛，对大肿瘤灵敏度高，小/扁平病变有限 |
| **膀胱镜检查** | **金标准**，可直视+取活检+切除小肿瘤 |
| **CT尿路造影（CTU）** | 评估上尿路和膀胱，用于进一步检查 |

## 筛查建议

### 筛查策略

| 人群 | 方案 |
|------|------|
| 一般人群 | **不推荐**常规筛查（发病率低） |
| 高危人群 | 每年1次尿常规+尿细胞学/NMP22 |
| 有症状者 | **无痛性肉眼血尿 → 立即膀胱镜检查** |
| 职业暴露者 | 每年1次尿常规+尿细胞学 |

### 筛查流程

```
高危人群
    → 每年 尿常规 + 尿细胞学/NMP22
    → 正常 → 每年复查
    → 异常 → 膀胱超声
             → 可疑 → 膀胱镜 + 活检
             → 正常 → 3-6个月后复查

无痛性肉眼血尿（任何人群）
    → 立即膀胱镜检查（不要延迟！）
```

## 预防建议

1. **戒烟**（最重要的预防措施，戒烟后风险逐年下降）
2. 职业暴露者严格做好**职业防护**
3. **多饮水**（每日>=2L），稀释尿液中致癌物浓度
4. 健康饮食，多吃新鲜蔬菜水果（富含维生素C和类胡萝卜素）
5. 避免长期使用含马兜铃酸的中草药

## 关键数据速查

| 指标 | 数据 |
|------|------|
| 吸烟归因风险 | ~50% |
| 尿细胞学灵敏度 | 13%-75%（低级别低，高级别高） |
| NMP22灵敏度 | 63%-98% |
| NMP22特异度 | 94%-99% |
| 最重要症状 | 无痛性肉眼血尿 |
