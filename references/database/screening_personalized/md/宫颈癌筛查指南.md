# 宫颈癌筛查指南

> 来源：李明珠 等. 中国子宫颈癌筛查指南（2023）
> 中国阴道镜和宫颈病理学会（CSCCP）等七大学（协）会
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **发病率趋势**：中国宫颈癌发病率从2.53/10万上升至11.34/10万（呈上升趋势）
- **死亡率**：约1.42/10万
- **病因**：高危型HPV持续感染是宫颈癌的**必要病因**（99.7%的宫颈癌与HPV相关）
- **消除目标**：WHO提出2030年消除宫颈癌战略（发病率<4/10万）

> 宫颈癌是目前唯一**病因明确、可预防、可消除**的癌症

## 高危对象

- 所有有性生活史的女性均应参与筛查
- 以下为更高风险因素：
  1. 多个性伴侣或性伴侣有多个性伴侣
  2. **早期性生活**（<18岁）
  3. 免疫抑制状态（**HIV感染者**、器官移植后、长期使用免疫抑制剂）
  4. 有宫颈癌家族史
  5. 吸烟女性（吸烟增加HPV持续感染风险）
  6. 多产次

## 筛查方法

### 推荐初筛方法：HPV核酸检测

本指南**首次推荐**高危型HPV核酸检测作为初筛首选方法！

| 方法 | 灵敏度 | 特异度 | 说明 |
|------|--------|--------|------|
| **HPV核酸检测**（初筛）| >95% | >90% | **首选推荐** |
| 宫颈细胞学（TCT） | 53%-81% | >90% | HPV检测不可及地区可用 |
| **HPV+TCT联合筛查** | 最高 | 最高 | 资源充足地区首选 |

### HPV初筛策略的优势
- 灵敏度显著高于细胞学
- 阴性预测值极高（HPV阴性 → 5年内宫颈癌风险<0.1%）
- 可自我采样（提高筛查覆盖率）

## 筛查建议

### 一般人群

| 年龄段 | 推荐方案 | 替代方案 |
|--------|----------|----------|
| **<25岁** | **不推荐筛查**（发病率极低） | — |
| **25-64岁** | 每 **5年1次** HPV核酸检测 | 每3年1次TCT |
| | 或每5年1次 HPV+TCT联合筛查 | |
| **>=65岁** | 如既往充分阴性记录*，**可终止筛查** | — |

> *充分阴性记录：过去10年内连续>=2次HPV阴性，或>=3次细胞学阴性，最近一次在3-5年内，且无非典型增生及以上病史

### HPV阳性结果管理

| 结果 | 管理 |
|------|------|
| **HPV16/18阳性** | 直接转诊**阴道镜**（无论细胞学结果） |
| **其他高危HPV阳性** + TCT正常 | 1年后复查 |
| **其他高危HPV阳性** + TCT异常 | 转诊**阴道镜** |
| HPV阳性持续1年以上 | 转诊阴道镜 |

### 特殊人群

| 人群 | 筛查建议 |
|------|----------|
| HIV感染者 | 有性生活第1年起，每年1次TCT；连续3次正常→每3年1次 |
| 子宫切除术后 | 因良性病变切除且无CIN2+史→停止筛查 |
| 妊娠期 | 按常规筛查（妊娠期不延迟） |
| 接种HPV疫苗后 | **仍按常规方案筛查**（疫苗不能覆盖所有高危型别） |

## 筛查流程

```
25-64岁女性
    → HPV核酸检测（首选）
    → HPV阴性 → 5年后复查
    → HPV16/18阳性 → 阴道镜
    → 其他高危HPV阳性 → TCT分流
        → TCT正常 → 1年后复查HPV
        → TCT异常 → 阴道镜
```

## HPV疫苗接种

### 疫苗类型

| 疫苗 | 覆盖型别 | 适用年龄 |
|------|----------|----------|
| 二价 | HPV16/18（70%宫颈癌） | 9-45岁 |
| 四价 | HPV16/18 + 6/11（尖锐湿疣） | 9-45岁 |
| 九价 | 7种高危+2种低危（90%宫颈癌）| 9-45岁 |

> 接种HPV疫苗后仍需定期筛查！疫苗不能替代筛查。

## 预防建议

1. **接种HPV疫苗**（最重要的一级预防措施，**9-14岁**接种效果最佳）
2. 保持安全性行为（使用安全套可降低HPV传播风险约70%）
3. **戒烟**（吸烟增加HPV持续感染和宫颈癌风险2-3倍）
4. 定期参加宫颈癌筛查
5. 男性伴侣包皮环切（可降低女性伴侣宫颈癌风险）
