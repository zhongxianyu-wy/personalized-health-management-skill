# 乳腺癌筛查指南

> 来源：沈松杰 等. 中国女性乳腺癌筛查指南（2022年版）
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **女性发病首位**：2022年新发35.72万例（女性恶性肿瘤第1位）
- **特点**：中国乳腺癌发病呈年轻化趋势，**绝经前患者比例高于欧美**
- **早诊率**：中国女性乳腺癌早期诊断率**显著低于欧美国家**
- **筛查覆盖率**：中国乳腺癌筛查覆盖率远低于发达国家（<30% vs >70%）

> 大量循证医学证据表明，乳腺癌筛查可提高早期诊断率、降低病死率。中国亟待加强和规范乳腺癌筛查。

## 高危对象

### 一般风险女性
- 除高危人群以外的所有适龄女性

### 高危人群

1. 有乳腺导管癌、小叶不典型增生或**小叶原位癌（LCIS）**病史者
2. **30岁前**接受过胸部放疗者（如霍奇金淋巴瘤放疗）
3. 有乳腺癌家族史

### 遗传性乳腺癌和卵巢癌综合征（HBOC）疑似患者

符合以下 **>=1项** 即为疑似患者：

- 家族中有 **BRCA1/BRCA2** 基因突变携带者
- 家族中有乳腺癌患者，发病年龄在 **45岁之前**
- 家族中有2个乳腺癌患者（1人双侧发病或2人单侧发病），发病年龄45-50岁
- 家族中有 **>=2个** 乳腺癌、卵巢癌、输卵管癌或原发性腹膜癌患者
- 家族中有 **男性乳腺癌** 患者
- 个人有乳腺癌/卵巢癌/输卵管癌/原发性腹膜癌史

## 筛查方法

### 方法比较

| 方法 | 适用人群 | 优势 | 局限 |
|------|----------|------|------|
| **乳腺X线摄影**（钼靶）| >=40岁 | 对钙化灶敏感，金标准 | 致密型乳腺灵敏度下降 |
| **乳腺超声** | 所有年龄 | 致密型乳腺优势，无辐射 | 对钙化不敏感 |
| **乳腺增强MRI** | 高危人群 | 最高灵敏度 | 价格贵，假阳性率高 |
| 乳腺体检 | 所有年龄 | 简便 | 灵敏度有限 |

### 致密型乳腺的筛查策略
- 中国女性乳腺以致密型为主（约占50-70%）
- 致密型乳腺（BI-RADS c/d型）：**X线+超声联合**可显著提高检出率
- 超声对中国女性乳腺癌筛查有特殊价值

## 筛查建议

### 一般风险女性

| 年龄段 | 筛查方案 |
|--------|----------|
| **<40岁** | **不推荐**常规影像学筛查（致密型乳腺+低发病率） |
| **40-70岁** | 每 **1-2年** 1次乳腺X线检查 |
| 致密型乳腺 | **X线 + 超声联合检查** |
| **>=70岁**（体健、预期寿命>=10年） | 维持筛查 |

### 高危人群

| 检查 | 频率 | 起始年龄 |
|------|------|----------|
| 乳腺X线 | **每年1次** | 40岁或更早 |
| 乳腺超声 | **每6-12个月** 1次 | 40岁或更早 |
| 乳腺体检 | **每6-12个月** 1次 | 40岁或更早 |
| 乳腺增强MRI | 必要时**每年1次** | 根据风险评估 |

### BRCA突变携带者
- **25-30岁起**开始乳腺筛查
- 每年1次 **乳腺MRI** + 乳腺X线（交替6个月间隔）
- 完成生育后可考虑**预防性双侧乳腺切除术**（降低风险90%+）
- 30岁起考虑**卵巢癌筛查**（CA125 + 阴道超声，每6-12个月）
- 35-40岁（完成生育）考虑**预防性卵巢输卵管切除术**

## 乳腺影像报告（BI-RADS）

| BI-RADS | 评估 | 恶性风险 | 管理 |
|---------|------|----------|------|
| 0 | 需进一步检查 | — | 补充影像 |
| 1 | 阴性 | 0% | 常规筛查 |
| 2 | 良性发现 | 0% | 常规筛查 |
| 3 | 可能良性 | <2% | 6个月短期随访 |
| **4A** | 低度可疑 | 2%-10% | **穿刺活检** |
| **4B** | 中度可疑 | 10%-50% | **穿刺活检** |
| **4C** | 高度可疑 | 50%-95% | **穿刺活检** |
| **5** | 高度提示恶性 | >95% | **穿刺活检** |
| 6 | 已证实的恶性 | 100% | 治疗 |

## 预防建议

1. 保持健康生活方式，远离烟酒，维持健康体重（BMI<25）
2. 适时生育（<30岁），**母乳喂养**（每哺乳12个月降低风险约4%）
3. 限制激素替代治疗（绝经后）
4. **定期参加乳腺筛查**
5. 高危人群考虑化学预防（他莫昔芬/雷洛昔芬，需医师评估）
6. BRCA突变携带者：完成生育后可考虑预防性手术
