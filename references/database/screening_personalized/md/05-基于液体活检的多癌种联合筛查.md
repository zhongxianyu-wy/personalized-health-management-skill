# 基于液体活检技术的多癌种联合筛查专家共识（2025版）

> 国家癌症中心 中国医学科学院北京协和医学院肿瘤医院
> 发表于《中华肿瘤杂志》2025年7月第47卷第7期
> 通信作者：陈万青

---

## 一、背景与意义

### 全球癌症筛查现状
- 预计到2050年，全球年新发癌症病例将超过3,500万（较2022年增加77%）
- 中国2022年新发482.5万例，死亡257.4万例
- 中国癌症发病率与世界平均水平接近，但**死亡率较高**（主要归因于较低的5年生存率）
- 目前仅少数癌种（结直肠癌、乳腺癌、宫颈癌）有标准筛查方法

### 传统单癌筛查的局限性
| 问题 | 具体表现 |
|------|----------|
| **侵入性强** | 结肠镜、胃镜等依从性差 |
| **覆盖面窄** | 仅少数癌种有筛查手段，大量癌种（胰腺癌、卵巢癌等）无筛查方案 |
| **准确性不足** | 假阴性/假阳性问题突出 |
| **成本效益低** | 单癌种逐一筛查效率低 |

### MCED的定义
**多癌种早期检测（Multi-Cancer Early Detection, MCED）**: 基于液体活检技术，通过检测血液等体液中的循环游离DNA（cfDNA）、循环肿瘤DNA（ctDNA）、RNA、蛋白质和代谢物等标志物，结合人工智能算法，实现**无创、高效的多癌种联合早期检测**。

---

## 二、技术原理与标志物

### 液体活检的生物学基础
肿瘤在生长过程中持续向血液中释放多种物质：

| 标志物类型 | 来源 | 检测内容 |
|-----------|------|----------|
| **ctDNA** | 肿瘤细胞凋亡/坏死/主动释放 | 基因突变、甲基化、片段化特征 |
| **cfDNA** | 体内全部细胞 | 甲基化模式、核小体印迹 |
| **CTCs**（循环肿瘤细胞） | 脱离原发灶的肿瘤细胞 | 细胞计数、分子特征 |
| **外泌体** | 肿瘤细胞分泌 | RNA、蛋白质 |
| **蛋白质** | 肿瘤分泌/宿主反应 | CA125、CEA、CA19-9等 |
| **代谢物** | 肿瘤代谢产物 | 代谢组学特征 |

### MCED核心技术路线

| 技术路线 | 原理 | 代表产品/研究 |
|----------|------|-------------|
| **cfDNA甲基化** | 检测癌细胞特异性甲基化模式（组织溯源） | Galleri (GRAIL)、国内多款 |
| **cfDNA突变+蛋白** | ctDNA突变特征 + 蛋白标志物联合 | CancerSEEK (DETECT-A) |
| **cfDNA片段化组学** | 核小体印迹分析（片段大小+末端序列） | DELFI |
| **多组学整合** | 甲基化+突变+片段化+蛋白+代谢物 | 综合AI算法 |

### 关键性能要求
| 指标 | 目标 |
|------|------|
| **特异性** | ≥98-99%（低假阳性是筛查的基本要求） |
| **灵敏度** | 按癌种和分期分层（I-II期需重点关注） |
| **组织溯源准确性** | ≥90%（准确预测肿瘤起源部位） |
| **PPV**（阳性预测值） | 取决于目标人群的癌症患病率 |

## 三、癌种选择标准

### MCED目标癌种应满足以下条件：
1. 癌症发病率或死亡率较高
2. 目前缺乏有效筛查手段，或现有筛查覆盖率不足
3. 已知在血液中可检测到ctDNA或其他循环标志物
4. 早期发现能显著改善预后（有有效的干预手段）

### 推荐纳入的癌种
| 优先级 | 癌种 | 理由 |
|--------|------|------|
| **高** | 胰腺癌、卵巢癌、肝癌、食管癌 | 目前无有效筛查，诊断时多为晚期 |
| **高** | 肺癌、结直肠癌、胃癌 | 高发病率，现有筛查覆盖率不足 |
| **中** | 乳腺癌、前列腺癌、膀胱癌 | 有现有筛查，MCED可作为补充 |
| **中** | 淋巴瘤、头颈部肿瘤 | 无常规筛查 |
| **低** | 甲状腺癌 | 预后好，过度诊断风险 |

## 四、研究设计与临床效用评估

### 研究设计阶梯
```
分析验证（Analytical Validation）
  → 临床验证（Clinical Validation，病例对照/队列）
    → 临床效用（Clinical Utility，RCT）
      → 真实世界应用（Real-World Implementation）
```

### 关键临床研究类型
| 研究类型 | 目的 | 举例 |
|----------|------|------|
| **病例对照研究** | 初步评估灵敏度/特异性 | CCGA (GRAIL) |
| **前瞻性队列研究** | 在目标筛查人群中验证 | DETECT-A, PATHFINDER |
| **RCT** | 评估是否降低癌症死亡率 | NHS-Galleri（英国，14万人） |
| **真实世界研究** | 评估在常规医疗中的表现 | — |

### 临床效用核心指标
- **癌症检出率**: 每万人筛查检出癌症数
- **假阳性率**: 不必要的后续检查
- **分期迁移**: 检出I-II期 vs III-IV期的比例变化
- **间期癌发生率**: 筛查间隔期内发生的癌症
- **最终目标**: **降低癌症特异性死亡率**

## 五、实施路径与推荐

### MCED适用人群
| 人群 | 推荐强度 | 说明 |
|------|----------|------|
| **50岁以上**（年龄风险） | 强推荐 | 大多数癌症发病率随年龄上升 |
| **癌症高危人群**（吸烟、家族史、职业暴露等） | 强推荐 | 获益可能更大 |
| **有症状者** | 推荐 | 作为辅助诊断而非筛查 |
| **45-50岁** | 推荐 | 可个体化考虑 |

### 阳性结果处理路径
```
MCED阳性
  → 根据组织溯源结果导向特定器官检查
    → CT/MRI/超声/PET-CT等影像学检查
      → 如找到可疑病灶 → 活检确诊
      → 如影像学阴性 → 短期密切随访（3-6个月复查MCED+影像）
```

### 专家推荐要点
1. MCED**不能替代**已证实有效且推荐的癌症筛查方法（如LDCT肺癌筛查、结直肠癌筛查）
2. MCED应作为现有筛查的**补充**，尤其是对缺乏有效筛查手段的癌种
3. MCED检测前需进行充分的**知情同意**
4. 阳性和阴性结果均需**标准化临床路径管理**
5. 鼓励MCED产品的**大规模前瞻性RCT验证**

---

## 六、挑战与展望

| 挑战 | 说明 | 解决方向 |
|------|------|----------|
| **生物学限制** | 部分早期肿瘤ctDNA释放量极少 | 多组学整合、提高检测灵敏度 |
| **假阳性** | 良性病变（如良性肿瘤、克隆性造血）导致 | 优化算法、联合多标志物 |
| **过度诊断** | 检出惰性肿瘤 | 生物学行为预测模型 |
| **卫生经济学** | 成本效益待验证 | 大规模RCT确定死亡率获益 |
| **标准化** | 不同检测平台结果可比性 | 参考物质、室间质评 |

> **中国大陆现状**: Galleri尚未在中国获批。国产MCED产品多处于临床验证阶段，需关注NMPA注册临床试验进展。
