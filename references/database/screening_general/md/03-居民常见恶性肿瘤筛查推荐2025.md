# 居民常见恶性肿瘤筛查和预防推荐（2025版）

> 来源：上海市抗癌协会 / 复旦大学附属肿瘤医院
> 自2018年起每年发布，2025版涵盖24个瘤种、6个专题、7大降低癌症风险建议

## 背景

- 中国每天超过 **1万人**罹患癌症，每8秒有1人被确诊
- 中国人一生患癌风险高达 **37%**
- WHO提出：接近一半的癌症可以预防，无法预防的癌症中，也有一半可以通过筛查早发现

## 2025版新增内容

- 新增癌种：**鼻咽癌、肾癌、子宫内膜癌**
- 修订内容：辐射与肿瘤、肿瘤心理、疫苗可预防癌症、胃癌、膀胱癌、儿童肿瘤、口腔癌

|------|------|

## 癌症筛查推荐时间表

| 年龄 | 推荐筛查项目 |
|------|-------------|
| 20+ | 宫颈癌筛查（有性生活后3年）、遗传性结直肠癌筛查、黑色素瘤自查 |
| 30+ | 宫颈癌筛查、乳腺癌高危人群筛查 |
| 40+ | 乳腺癌筛查、肺癌高危人群筛查、胃癌高危人群筛查、肝癌高危人群筛查 |
| 45+ | 结直肠癌一般人群筛查、食管癌高危人群筛查 |
| 50+ | 前列腺癌筛查、胰腺癌高危人群筛查 |
| 55+ | 肺癌筛查（吸烟≥30包年） |

## 七大降低癌症风险建议

1. **不吸烟/戒烟** — 避免二手烟
2. **健康饮食** — 多吃全谷物、蔬菜、水果、豆类，限制红肉和加工肉
3. **保持健康体重** — 避免肥胖，坚持体育锻炼
4. **限制饮酒** — 最安全的饮酒量是0
5. **防晒** — 避免过度日晒，使用防晒措施
6. **疫苗接种** — HPV疫苗、乙肝疫苗
7. **定期体检** — 按推荐进行癌症筛查

## 新兴筛查方式：多癌种早期检测（MCED）

基于液体活检的MCED技术（如ctDNA甲基化检测）正成为传统筛查的补充手段，尤其对以下**无标准筛查方法**的癌种具有独特价值：

| 癌种 | 标准筛查 | MCED价值 |
|------|----------|----------|
| 食管癌 | 无 | 填补筛查空白 |
| 胰腺癌 | 无 | 填补筛查空白 |
| 卵巢癌 | 无（仅高危人群）| 填补筛查空白 |

> **产品参考**：[[../吉早安/产品概述|吉早安]] — 基于NGS靶向甲基化测序，一管10ml血筛查8种高发癌症，特异性99.0%，敏感性81.9%。详见[[../吉早安/癌种覆盖|癌种覆盖]]。
