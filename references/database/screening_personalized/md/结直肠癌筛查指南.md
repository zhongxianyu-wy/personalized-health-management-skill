# 结直肠癌筛查指南

> 来源：陈万青 等. 中国结直肠癌筛查与早诊早治指南（2020，北京）
> 国家癌症中心
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **发病第二位**：2022年新发51.71万例
- **中标发病率**：男性 15.6/10万，女性 11.6/10万，死亡率 10.5/10万
- **趋势**：中国结直肠癌发病率持续上升，城市高于农村
- **特点**：大多数结直肠癌由**腺瘤性息肉**经过5-10年逐步发展而来（腺瘤-癌序列）

> 从腺瘤到癌的演变过程提供了宝贵的筛查窗口期

## 高危对象

### 一般风险人群
1. **45岁以上** 无症状人群

### 高风险人群（满足任一即视为高风险）
2. **40岁以上** 有两周肛肠症状*的人群
3. 长期患有**溃疡性结肠炎**的患者
4. 结直肠癌术后/大肠腺瘤治疗后的人群
5. 有慢性胆道疾病史或胆囊切除史的人群
6. 有不良生活事件史（近20年内重大精神创伤）的人群

### 遗传高风险人群
7. 有结直肠癌家族史人群的直系亲属（一级或二级亲属）
8. 遗传性结直肠癌家族直系亲属，年龄超过 **20岁**（林奇综合征、FAP等）

> *两周肛肠症状：大便习惯改变（便秘/腹泻）、大便形状变细、大便性质改变（便血/黏液便）、腹部固定部位疼痛，持续>=2周

## 筛查方法

### 方法一览

| 筛查方法 | 频率 | 灵敏度 | 特异度 | 特点 |
|----------|------|--------|--------|------|
| 粪便免疫化学检测（**FIT**）| 每年1次 | ~88% | ~95% | 首选初筛，简便经济 |
| 粪便多靶点 **FIT-DNA** 检测 | 每1-3年1次 | ~95% | ~98% | 灵敏度更高，价格贵 |
| 血液循环肿瘤DNA（ctDNA）检测 | 每1-3年1次 | 85-90% | 95%+ | 新兴无创方法 |
| **结肠镜** | 每5-10年1次 | 金标准 | — | 可同时活检/切除息肉 |

### 结肠镜筛查效能
- 结肠镜可降低结直肠癌发病率 **76%**（切除腺瘤性息肉）
- 可降低结直肠癌死亡率 **53%**

## 筛查建议

### 一般人群（45-75岁）

| 策略 | 方案 |
|------|------|
| **初筛策略** | 每年1次FIT → 阳性者行结肠镜 |
| **直接筛查** | 每5-10年1次结肠镜（推荐首选） |
| **联合筛查** | FIT + ctDNA 联合可提高准确度 |

> 粪便/血液筛查阳性者 → **必须立即行肠镜检查**，不宜延迟超过6个月

### 有结直肠癌家族史（40岁起）
- **高风险家族史**（一级亲属患高级别腺瘤/癌，发病<60岁，或>=2位一级亲属患癌）：
  - 每年FIT，每**5年至少1次**肠镜
  - 起始年龄：40岁 or 比家族最小发病者年龄小10岁
- **一般家族史**（仅1位一级亲属患癌，发病>=60岁）：
  - 每年FIT，每**10年至少1次**肠镜

### 遗传性结直肠癌家族成员
- **林奇综合征**：20-25岁起，每1-2年1次肠镜
- **FAP（家族性腺瘤性息肉病）**：10-12岁起，每年1次乙状结肠镜
- 先进行**遗传咨询和基因检测**

### 停止筛查
| 年龄段 | 建议 |
|--------|------|
| 76-85岁 | 体健、预期寿命>10年者可继续 |
| **>85岁** | 不推荐继续筛查 |

## 筛查流程

```
45-75岁一般人群
    → FIT（每年1次）或 结肠镜（每5-10年）
    → FIT阳性 → 结肠镜
                → 无异常 → 5-10年复查
                → 腺瘤 → 根据病理决定随访间隔（1-5年）
                → 癌 → 治疗
```

## 预防建议

1. 坚持体育锻炼，避免肥胖（BMI<25）
2. 健康膳食：增加**粗纤维、新鲜水果蔬菜、全谷物**，限制红肉和加工肉
3. 限制饮酒，戒烟
4. 老年人可咨询医师后服用低剂量阿司匹林（可能同时降低心脑血管风险）
5. 补充**钙和维生素D**可能有保护作用
