# 套餐三档与风险驱动

> 用途：v2.0 单一报告「组合套餐+预算推荐」模块。LLM 参考本文 + 个体 snapshot 风险，动态拼装三档套餐。
> 边界：价格为运营性区间（非临床证据，不进概率数学）；吉早安性能数值引用 `cancerrisk/json/detection_performance.json`（不编造）。

## 一、三档定义（风险驱动 + 吉早安角色）

### 档1 · 风险靶向聚合档
- **组成规则**：包含个体**所有高风险项目**（癌症后验 >1% 或 健康总结评估较严重）；若不足 5 项，用**中等风险项目**（后验 0.5%-1%）补齐至 5 项；仍不足则用第三档（缺口）项目按优先级补齐。
- **定位**：聚焦当前最紧急的风险，最小必要检查集。

### 档2 · 全面覆盖档（推荐）
- **组成规则**：**全面型**，覆盖全部三档风险——高风险项 + 中等风险项 + 筛查缺口项 + 基础必检（一般检查/血尿便常规/肝肾脂糖/心电/腹部超声等）。
- **定位**：一次性全面排查，平衡覆盖与预算。

### 档3 · 吉早安替换/弥补档（两种价格策略，可各一卡或合并展示）
- **① 替换型**：将吉早安目标癌症（肺/结直肠/胃/肝/食管/胰腺/乳腺/卵巢）涉及的**专项筛查项目替换为吉早安**——一管血 cfDNA 多癌早筛替代多个单项，提升便捷性、降低创伤（替代肠镜/胃镜等侵入项时尤甚）。适合畏惧侵入检查或追求便捷者。
- **② 弥补型**：在档2 全面覆盖基础上**额外加吉早安**——标准筛查 + 多癌早筛双重覆盖，补足无标准筛查癌种（胰腺/卵巢/肝/食管）。适合追求最全面深入者。
- **定位**：吉早安作为便捷替代或深度补充（对齐《液体活检多癌筛查专家共识 2025》：MCED 是现有筛查的补充/平替，不能替代 LDCT/肠镜等已证实有效的筛查）。

> 吉早安覆盖 8 癌 + 性能（sens **81.9%** / spec **99.0%**）引用 `cancerrisk/json/detection_performance.json` 综合行 `jizaoan_multi_cancer_screening_overall`，不编造。阳性→升档+溯源影像；阴性→LR⁻ 降风险评级。

## 二、风险驱动逻辑（读 `artifacts/snapshot_risk.json`）

按个体癌症风险分级（snapshot_risk.json 的 `risk_tier`）驱动推荐：

| 条件 | 推荐档 |
|---|---|
| 全部癌种 low + 年龄<45 + 无一级亲属癌症家族史 | 档1 |
| 任一癌种 medium / 年龄≥45 / 家族史阳性 / 任一异常指标 | **档2（默认推荐）** |
| 任一癌种 high/very_high / 遗传突变 / 吉早安阳性 | 档3 |

> 报告中三档**同时展示**，标注哪档为「该个体首选」并给出理由（偶联 snapshot 风险数据，不编造）。

## 三、吉早安（cfDNA 多癌早筛）角色

**定位**：癌症风险检测能力位的**平替/补充**，非独立推荐卡片。对齐《液体活检多癌筛查专家共识2025》—— MCED 是现有筛查的**补充**，不能替代已证实有效的筛查（LDCT/结直肠等）。

- **覆盖 8 癌**：肺/结直肠/胃/肝/食管/胰腺/乳腺/卵巢（参 `detection_performance.json`，source_id=`jizaoan_white_paper_v2026`）
- **价值**：补充**无标准筛查癌种**（胰腺/卵巢/肝/食管）的检测
- **结果处理**（双路径引擎已实现）：
  - **阳性** → 升档3 + 按组织溯源做高精度影像（CT/MRI/PET-CT）→ 可疑病灶活检
  - **阴性** → LR⁻ 路径降低覆盖癌种风险评级（报告展示「可降低目前高风险癌症多少风险评级」）
- **性能**：灵敏度/特异度引用 `detection_performance.json`，不编造

## 五、套餐拼装约束（偶联数据，PUA）

- 三档内容严格偶联上述数据：项目来自筛查指南、价格来自 `pricing/md/08`、风险来自 `snapshot_risk.json`。
- **复合价格计算**：套餐价格区间 = 所含各项目参考价之和。例：进阶全面档 ≈ LDCT(500)+胃肠镜(1800)+HbA1c(70)+颈动脉超声(200)+吉早安(1980-2980)+… → 约 3000-8000。每档价格是「项目复合」而非拍脑袋。
- **报告模版对齐**：报告 `package-grid` 三档卡片（参 `temp/html-preview-10.html`）= 名称 + 价格区间 + 包含项目 + 定位说明。高危场景（遗传/吉早安阳性＝档3）三档实例价格更高（模版：遗传核心 2000-3500 / 精准型 4000-9000 / 长程型 6000-12000，因含 MRI/吉早安等贵项目），通用低风险场景则用本文件档1/档2 区间。
- 价格为**区间**，统一标注「以机构报价为准」。
- **长期健康生活干预 + 专科建议**：严格按报告模版（参 `07-肿瘤预防与健康管理.md` + `化验指标判读规则.md`）。
- 不得编造项目、价格、性能数值（PUA 禁止行为）。
