# 前列腺癌筛查指南

> 来源：赫捷 等. 中国前列腺癌筛查与早诊早治指南（2022，北京）
> 国家癌症中心
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **发病率**：16.6/10万（中国城市地区上升趋势明显）
- **趋势**：中国前列腺癌发病率呈**快速上升**趋势，与生活方式西化、老龄化相关
- **特点**：中国前列腺癌初诊晚期比例（~30%）高于欧美国家

> 前列腺癌是发达国家男性最常见的癌症，随着中国老龄化和生活方式改变，防控形势日益严峻

## 高危对象

| 风险等级 | 人群 | 起始筛查年龄 |
|----------|------|-------------|
| 一般风险 | 所有男性 | **50岁** |
| 较高风险 | 有前列腺癌家族史（一级亲属）| **45岁** |
| 高风险 | BRCA2突变携带者 | **40岁** |
| 特殊人群 | 有乳腺癌家族史的男性 | 45岁 |

## 筛查方法

### PSA检测（首选初筛方法）

| 指标 | 说明 |
|------|------|
| 灵敏度 | **89%-100%**（PSA>4 ng/mL） |
| 特异度 | **13%-77%**（较低，容易假阳性） |
| 方法 | 抽血检测，简便经济 |

### PSA结果解读

| PSA值 | 风险评估 |
|-------|----------|
| **<4 ng/mL** | 正常，按年龄和风险定期复查 |
| **4-10 ng/mL** | 灰区，需结合PSA密度、游离/总PSA比值等判断 |
| **>10 ng/mL** | 高风险，建议进一步检查 |

### 进一步检查方法

| 方法 | 说明 |
|------|------|
| 直肠指检（**DRE**）| 基础检查，可触及前列腺硬结 |
| 多参数 **MRI**（mpMRI） | PSA异常者的首选进一步检查，PI-RADS评分 |
| **前列腺穿刺活检** | 确诊金标准（经直肠或经会阴） |
| **PSMA-PET/CT** | 用于高危患者的精准分期 |

### 筛查新策略：PSA → mpMRI → 靶向穿刺
- 传统流程：PSA异常 → 系统性穿刺（10-12针） → 可能过度诊断
- 现代流程：PSA异常 → mpMRI → PI-RADS>=3 → 融合靶向穿刺 → **减少不必要的穿刺**

## 筛查建议

### 一般风险人群
- **50岁起**，每1-2年1次PSA检测
- 预期寿命<10年者不推荐筛查

### 高风险人群
- **45岁起**（有家族史），**40岁起**（BRCA2突变）
- 每年1次PSA检测
- PSA异常 → mpMRI → 必要时穿刺

### 停止筛查
- 年龄>75岁 **且** 预期寿命<10年 → 停止筛查

## PSA筛查的获益与争议

### 获益
- 可降低前列腺癌特异性死亡率约 **20-30%**
- 早期发现可治愈的前列腺癌

### 风险
- **过度诊断**：可能检出永远不会进展的惰性前列腺癌
- **假阳性**：导致不必要的穿刺和焦虑
- 穿刺风险：感染、出血、尿潴留

> **医患共同决策**：筛查前应与医师充分沟通获益和风险

## 预防建议

1. 健康饮食：减少动物脂肪，增加番茄（番茄红素）、豆类、绿茶
2. 适度体育锻炼，维持健康体重
3. 高危人群定期PSA检测
4. 有遗传风险者进行BRCA等基因检测
