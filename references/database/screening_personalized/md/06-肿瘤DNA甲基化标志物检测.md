# 肿瘤DNA甲基化标志物检测及临床应用专家共识（2024版）

> 中国抗癌协会肿瘤标志专业委员会
> 发表于《中国癌症防治杂志》2024年4月第16卷第2期

---

## 一、DNA甲基化标志物概述

### 什么是DNA甲基化
DNA甲基化是指DNA甲基转移酶（DNMTs）将甲基添加到CpG序列中胞嘧啶的5'碳位，形成5-甲基胞嘧啶的过程。成簇的CpG称为CpG岛（CGIs），主要位于基因启动子和第一外显子区域。

### 作为肿瘤标志物的优势
| 优势 | 说明 |
|------|------|
| **更早期** | 甲基化异常发生在肿瘤形成的初始阶段（癌前病变期即可检出） |
| **更无创** | 可通过血液、痰液、尿液、粪便等非侵入性样本检测 |
| **更精准** | 组织特异性甲基化模式可追溯肿瘤起源 |
| **多场景** | 覆盖筛查→诊断→治疗→复发监测全周期 |

### 检测样本类型
| 样本 | 检测组分 | 应用场景 |
|------|----------|----------|
| **血液（血浆）** | cfDNA/ctDNA | 多癌种筛查、MRD监测 |
| **组织**（新鲜/FFPE） | 细胞基因组DNA | 辅助诊断、伴随诊断、预后评估 |
| **粪便** | 脱落细胞DNA | 结直肠癌筛查 |
| **尿液** | 脱落细胞DNA | 膀胱癌/尿路上皮癌筛查 |
| **痰液/灌洗液** | 脱落细胞DNA | 肺癌筛查 |

### 血液甲基化检测三大组分
| 组分 | 来源 | 特点 |
|------|------|------|
| **PBMC DNA** | 淋巴细胞+单核细胞 | 体细胞代表性，受细胞活性影响 |
| **cfDNA** | 全身细胞凋亡/坏死/主动释放 | 含肿瘤+非肿瘤DNA，半衰期~2h |
| **ctDNA** | 肿瘤细胞释放 | 肿瘤特异性，丰度低 |

## 二、检测技术与方法

### 2.1 DNA提取与纯化
| 要点 | 推荐 |
|------|------|
| **方法选择** | 离心柱法或磁珠法；富含cfDNA样本推荐两步离心法 |
| **分区处理** | cfDNA与细胞/组织样本分区处理（避免基因组污染） |
| **自动化** | 推荐自动化提取设备（减少批间差异） |
| **保存要求** | 分离后-70℃及以下长期保存，避免反复冻融（≤3次） |

### 样本前处理关键注意事项
| 样本类型 | 注意事项 |
|----------|----------|
| **血液（cfDNA）** | 不可使用肝素抗凝！推荐EDTA或枸橼酸；采集量 ≥2×10mL；采血后4-6h内分离；分离前不可冻存全血 |
| **含防腐剂采血管** | 建议3d内分离（可延迟处理） |
| **粪便** | 推荐含防腐剂保存液，减少室温运输时间 |
| **组织** | 低温保存或液氮速冻；FFPE切片8-10μm |
| **拭子** | 专用采集器，室温风干保存≤3d |

### 2.2 转化方法
| 方法 | 原理 | 特点 |
|------|------|------|
| **亚硫酸氢盐转化** | 未甲基化C→U→T（PCR），甲基化C不变 | **金标准**，应用最广泛；DNA降解率高 |
| **酶法转化** | TET氧化+硼烷还原，甲基化C→DHU→T | DNA损伤小，转化效率待优化 |
| **MSRE-PCR** | 甲基化敏感性限制酶切割 | 操作简单，DNA损失少；酶切效率需优化 |

### 2.3 检测平台
| 平台 | 通量 | 应用 |
|------|------|------|
| **实时荧光定量PCR（qPCR）** | 低（1-数个基因） | 单癌种筛查、辅助诊断（最常用） |
| **NGS** | 高（全基因组/靶向panel） | MCED多癌筛查、MRD |
| **核酸质谱** | 中 | 多基因panel甲基化 |

## 三、临床应用

### 3.1 肿瘤早期筛查
| 应用 | 代表产品/标志物 | 样本 | 状态 |
|------|---------------|------|------|
| 结直肠癌筛查 | **SEPT9** 基因甲基化（血浆） | 血液 | NMPA获批 |
| 结直肠癌筛查 | **SDC2** 基因甲基化（粪便） | 粪便 | NMPA获批 |
| 结直肠癌筛查 | 多靶点粪便FIT-DNA（Cologuard） | 粪便 | FDA获批 |
| 肺癌筛查 | SHOX2/RASSF1A/PTGER4 甲基化 | 血液/痰液 | NMPA获批 |
| 肝癌筛查 | 多个甲基化panel | 血液 | 临床验证中 |
| 多癌筛查 | 甲基化+多组学（Galleri） | 血液 | FDA突破性器械，临床验证 |
| 膀胱癌 | 尿液脱落细胞甲基化 | 尿液 | 在研 |
| 宫颈癌/子宫内膜癌 | 甲基化panel | 宫颈脱落细胞 | 辅助HPV阳性分流 |

### 3.2 辅助诊断与伴随诊断
- **胶质瘤**: MGMT启动子甲基化 → 替莫唑胺（TMZ）敏感性预测
- **卵巢癌/乳腺癌**: BRCA1启动子甲基化 → PARP抑制剂敏感性
- **结直肠癌**: MLH1启动子甲基化 → 散发MSI-H CRC的鉴别
- **肺癌**: 多个基因甲基化panel → 辅助鉴别良性vs恶性结节

### 3.3 MRD检测与复发监测
- 甲基化标志物可用于**根治性治疗后的MRD评估**
- 相比ctDNA突变检测，甲基化检测**不依赖个体化panel设计**
- 甲基化水平升高可早于影像学复发数月至一年
- 需注意良性病变和克隆性造血可能导致的假阳性

### 3.4 预后评估
- 特定基因低甲基化（如LINE-1） → 可能提示预后不良
- 多个基因甲基化panel → 综合预后分层

## 四、专家共识推荐要点

| 编号 | 推荐内容 | 推荐强度 |
|------|----------|----------|
| 1 | 样本采集后应尽可能减少转运耗时，及早分离检测组分 | 强推荐 |
| 2 | 血液样本应避免溶血，不可使用肝素抗凝 | 强推荐 |
| 3 | 检测游离DNA时采集量应充足（≥20mL全血），尽早两步离心分离无细胞血浆 | 强推荐 |
| 4 | DNA甲基化可作为现有癌症筛查方法的**补充**手段，尤其适用于拒绝侵入性筛查的人群 | 推荐 |
| 5 | 甲基化检测用于MRD监测时，**不应单独用于治疗决策**，需结合影像和临床评估 | 推荐 |
| 6 | 甲基化检测产品使用前应明确其**临床效用**（灵敏度/特异性/PPV/NPV） | 强推荐 |
| 7 | 检测结果解读应由有经验的临床医生进行，结合患者的整体临床信息 | 推荐 |
