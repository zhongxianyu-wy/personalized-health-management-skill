# 健康体检基本项目专家共识（2022）

> 来源：健康体检基本项目专家共识（2022年版）

## 共识概述

该共识为健康体检机构提供了标准化的体检项目体系，采用 **"1+X"模式**：

- **"1"** = 基本体检项目（必选，适用于所有体检者）
- **"X"** = 个性化体检项目（根据个体风险因素选择）

## 基本体检项目（"1"）

### 健康体检自测问卷
- 个人健康史、家族史、生活方式、心理状态等

### 体格检查
- 身高、体重、腰围、血压
- 内科、外科、眼科、耳鼻喉科、口腔科、妇科

### 实验室检查
- 血常规、尿常规、粪便常规+隐血
- 肝功能、肾功能、血脂、血糖
- 肿瘤标志物（基础）

### 辅助检查
- 心电图
- 胸部X线/低剂量CT
- 腹部超声
- 妇科超声（女性）

## 个性化体检项目（"X"）

根据以下因素进行个性化添加：
- **年龄**：不同年龄段重点不同
- **性别**：男性前列腺、女性乳腺/宫颈
- **家族史**：肿瘤家族史者加强筛查
- **个人史**：吸烟、饮酒、职业暴露等
- **既往疾病史**：慢性病患者针对性检查
- **体检发现**：既往体检异常项目追踪

## "1+X"模式的意义

| 维度 | 内容 |
|------|------|
| 标准化 | 保证基础质量，覆盖常见疾病 |
| 个性化 | 避免过度检查和漏检 |
| 科学性 | 基于循证医学证据 |
| 经济性 | 优化医疗资源配置 |
