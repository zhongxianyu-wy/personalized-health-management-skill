# 异常指标复查推荐（异常 → 复查项目 + 价格）

> 用途：v2.0 单一报告「异常指标复查推荐」（复查三级中的「重要检查/持续管理」来源之一）。当化验指标异常（grade_level≥1），偶联本表给出具体复查/加项项目 + 目的 + 价格参考。
> 数据偶联：异常分级来自 `化验指标判读规则.md`；价格来自 `pricing/md/08-体检筛查项目价格参考.md`（区间，以机构报价为准）；不进概率数学。
> 应用模式：LLM 参考（MD，非脚本查表）。

## 映射表（异常指标 → 推荐复查/加项）

| 异常触发 | 推荐复查/加项 | 目的 | 价格参考(元) |
|---|---|---|---|
| **FPG 6.1-6.9 / HbA1c 5.7-6.4**（糖尿病前期）| OGTT + HbA1c 复查 | 确认糖代谢状态 | OGTT 180 / HbA1c 70 |
| **FPG≥7.0 / HbA1c≥6.5**（糖尿病）| OGTT + 并发症筛查：UACR+eGFR / 眼底 / 神经传导 | 确诊+并发症评估 | UACR 60 / 眼底 / 神经传导 |
| **TC/TG/LDL-C 升高**（血脂异常）| 血脂全套 + 载脂蛋白A1/B+Lp(a) + ASCVD 评估 + 颈动脉超声 | 心血管风险分层 | 颈动脉超声 200 / 载脂蛋白 150 / 超声心动 142 |
| **LDL-C 显著升高(≥4.1)** | 心内科/内分泌科就诊 + ASCVD 评估 | 高危血脂干预 | 同上 |
| **ALT/AST/GGT 升高**（肝功）| 肝炎病毒筛查(HBsAg/HCV-Ab) + 肝脏超声 ± FibroScan | 排查病毒性肝炎/脂肪肝/纤维化 | 肝炎筛查 100-200 / 肝超声 100 / FibroScan 300 |
| **ALT/AST 显著升高(>5倍ULN)** | 消化/肝病科就诊 + 肝炎病毒 + 药物性肝损排查 | 急性肝损伤 | 肝炎筛查 + 腹部超声 |
| **CREA/BUN 升高**（肾功）| eGFR 计算 + 尿微量白蛋白/UACR + 肾超声 | 肾功能评估 | UACR 60 / 肾超声（含腹部） |
| **UA 升高**（高尿酸）| 关节超声(有症状时) + 泌尿超声 | 痛风/结石风险 | 超声 / 风湿科随诊 |
| **TSH 异常**（甲亢/甲减/亚临床）| FT3/FT4 + TPOAb + 甲状腺及颈部淋巴结超声 | 甲功定性 | 甲状腺超声 120 / TPOAb / FT3/FT4 |
| **WBC 减少或升高** | 复查 + 外周血涂片 + 必要时血液科 | 排除感染/血液病 | 血涂片 |
| **HGB 降低**（贫血）| 铁代谢/叶酸/VitB12 + 消化道排查(便潜血/胃肠镜) | 贫血病因 | 铁代谢 / 胃肠镜 1000-1800 |
| **PLT 减少或升高** | 复查 + 血液科评估 | 出血/血栓风险 | / |
| **尿蛋白阳性(≥1+)** | UACR + eGFR + 肾内科 | 肾损伤评估 | UACR 60 |
| **尿糖阳性** | 血糖 + HbA1c | 糖代谢(超肾糖阈) | HbA1c 70 |
| **尿隐血阳性** | 尿沉渣镜检 + 泌尿系统超声 | 真性血尿排查 | 泌尿超声（含腹部） |

## 复查三级归类（与「癌症风险分层与复查规则.md」协同）

LLM 按异常严重度（grade_level + 临床意义）将上表复查项归入报告复查三级：
- **优先执行**（grade_level≥2 或显著升高，如糖尿病/显著肝损/贫血）：立即就诊+确诊检查（最多 3-5 项）
- **重要检查**（grade_level=1，需病因排查）：肝炎筛查/颈动脉超声/OGTT 等（过多则右侧续排）
- **持续管理**（慢病监测，如糖尿病前期/血脂边缘）：定期复查 HbA1c/血脂 + 生活方式

## 与套餐三档的关系

异常复查项可并入**套餐档位**（参 `套餐三档与风险驱动.md`）：
- 基础精准档：仅关键异常 1-2 项复查
- 进阶全面档：代谢心血管全套（HbA1c/OGTT/颈动脉/超声心动）+ 肝肾病因筛查
- 深度早筛档：进阶 + 高精度（如 FibroScan/胃肠镜）+ 并发症全面评估

价格区间由所含项目参考价之和得出（参 `pricing/md/08`），标注「以机构报价为准」。

## 约束（PUA）
- 异常分级必须来自 `化验指标判读规则.md`（不编造阈值）
- 复查项目+价格必须偶联本表 + `pricing/md/08`（不编造）
