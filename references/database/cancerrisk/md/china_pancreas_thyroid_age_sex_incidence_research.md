# 中国人群胰腺癌与甲状腺癌年龄-性别基线发病率调研

生成日期：2026-05-25  
调研目标：补齐 `database_new/evidence_store/ontology/cancer_age_sex_priors.json` 中胰腺癌与甲状腺癌按性别、年龄分层的人群基线年发病率。  
单位：年发病率 `/100,000`；年概率 = 年发病率 / 100000。

## 结论

- 最适合直接入库的可机读来源是 IARC Global Cancer Observatory / Cancer Today 的 GLOBOCAN 2022 v1.1（API 数据版本 `2022ps`）。
- 中国国家癌症中心 2022 论文提供中国癌症总体负担、估计方法和癌种排序，是权威背景来源；但网页正文不直接给出所有癌种的 5 岁年龄组 × 性别完整表。
- 当前数据库中胰腺癌已有 `model_estimated_nccr2011` 版本，建议用 GLOBOCAN 2022 API 的年龄-性别分层值替换或新增为 `globocan_2022_gco_api` 来源。
- 当前数据库中甲状腺癌在 `missing_priors` 中，建议新增 `thyroid_cancer`，并采用同一 GLOBOCAN 2022 API 来源。

## 数据来源

1. IARC Global Cancer Observatory, Cancer Today, GLOBOCAN 2022 v1.1 / API version `2022ps`。API base: `https://gco.iarc.fr/gateway_prod/api/globocan/v3/2022ps/`。
2. Han BF, Zheng RS, Zeng HM, et al. *Cancer incidence and mortality in China, 2022*. Journal of the National Cancer Center, 2024. PubMed: https://pubmed.ncbi.nlm.nih.gov/38468501/；全文页: https://www.sciencedirect.com/science/article/pii/S2667005424000061。

## API 口径

- 国家：中国，GCO country code `156`。
- 胰腺癌：GCO cancer code `13`，ICD-10 `C25`。
- 甲状腺癌：GCO cancer code `32`，ICD-10 `C73`。
- 男性：sex code `1`；女性：sex code `2`。
- 年龄组：`ages_group=0_0` 到 `17_17`，对应 0-4 到 85+。
- 示例：`https://gco.iarc.fr/gateway_prod/api/globocan/v3/2022ps/data/rate/0/1/156/13/?ages_group=10_10` 返回中国男性 50-54 岁胰腺癌发病率。

## 入库建议

- 胰腺癌：保留旧 `model_estimated_nccr2011` 作为历史参考，新增或替换为 `source_id = globocan_2022_gco_api`。
- 甲状腺癌：新增 `thyroid_cancer` 条目，并从 `missing_priors` 移除 `thyroid_cancer`。
- 若系统只支持 30-75 岁锚点，使用“与当前数据库年龄锚点对齐的建议值”章节即可。
- 若后续希望覆盖儿童/青少年/85+，建议把 prior lookup 扩展为 0-4 到 85+ 完整年龄组。
