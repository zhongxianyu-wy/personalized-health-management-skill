# 肺癌筛查指南

> 来源：赫捷 等. 中国肺癌筛查与早诊早治指南（2021，北京）
> 国家癌症中心/国家肿瘤临床医学研究中心
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **发病首位**：肺癌是中国发病率和死亡率最高的恶性肿瘤
- **中标发病率**：男性 61.5/10万，女性 29.4/10万
- **全球占比**：中国肺癌发病数和死亡数分别占全球的 **37.0%** 和 **39.8%**（GLOBOCAN 2020）
- **临床分期**：2012-2014年，中国III-IV期肺癌占比达 **64.6%**（多数发现即为晚期）
- **5年生存率**：I期 **55.5%**，IV期仅 **5.3%**；总体5年生存率不超过 **20.0%**

> 中国肺癌年龄标化5年生存率在2003-2015年间略有上升，但仍不超过20.0%

## 高危对象

**40岁以上**，符合以下 **>=1种** 情况者：

1. **吸烟>=20年包**（每天1包持续20年或每天2包持续10年），包括戒烟时间不足15年
2. 被动吸烟（二手烟暴露）
3. 职业暴露史（**石棉、铍、铀、氡**等接触者）
4. 恶性肿瘤病史（淋巴瘤、头颈部癌症或与吸烟有关的癌症）
5. 家族史：**一级亲属**罹患肺癌
6. 慢性肺部疾病史（COPD、弥漫性肺纤维化、肺结核等）

## 筛查方法

### 推荐技术：低剂量螺旋CT（LDCT）

| 参数 | 说明 |
|------|------|
| CT类型 | **64排或以上** 多排螺旋CT |
| 扫描范围 | 肺尖至肋膈角尖端水平 |
| 剂量 | 低剂量（降低辐射暴露） |
| 灵敏度 | **93.8%**（95%CI：90.6%-96.3%） |
| 特异度 | **78.2%** |

### 为什么不用X线胸片？
- 胸部X线灵敏度仅71.2%，容易漏诊早期肺癌
- LDCT相比X线可降低肺癌死亡率约 **20%**（基于NLST研究）

### 筛查频率
- 高危人群：**每年1次** LDCT
- 基线CT阴性者：继续每年1次
- 结节随访：根据结节类型和大小决定间隔（3-12个月）

## 肺结节管理

### 结节分类与处理

| 结节类型 | 特征 | 管理策略 |
|----------|------|----------|
| **磨玻璃结节（GGN）** | 密度略高于正常肺组织 | <5mm无需随访；>=5mm定期CT随访 |
| **亚实性结节** | 磨玻璃+实性成分 | 根据实性成分大小决定随访或活检 |
| **实性结节** | 均匀软组织密度 | <8mm定期随访；>=8mm评估活检 |
| **多发结节** | 多个结节同时存在 | 以风险最高的结节为准制定方案 |

### 评估维度
- 影像学特征：大小、形态、密度、边缘特征（分叶、毛刺等）
- 生物学评估：ctDNA、甲基化等液体活检辅助评估
- 推荐**影像学+生物学多维度精准评估**

## 筛查流程

```
高危人群评估 → LDCT筛查 → 阴性 → 每年1次LDCT
                            → 阳性 → 肺结节评估
                                     → 低风险 → 定期随访（3-12个月）
                                     → 高风险 → 多学科会诊（MDT）
                                              → 进一步检查/手术
```

## LDCT筛查的获益与风险

### 获益
- 降低肺癌死亡率约20%
- 早期发现提高治愈率：I期5年生存率55.5%

### 风险
- **假阳性**：可能导致不必要的进一步检查
- **过度诊断**：可能检出永远不会进展为临床癌症的惰性病变
- **辐射暴露**：LDCT每次辐射剂量约1.5 mSv（常规CT约8 mSv）

## 预防建议

1. **戒烟**（最重要的预防措施），避免二手烟暴露
2. 有职业暴露危险者做好防护
3. 有呼吸系统疾病者及时规范治疗
4. 健康饮食，增加新鲜蔬菜水果摄入
5. 参加规范的肺癌筛查
