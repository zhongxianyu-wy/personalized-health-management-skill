## 入库字段建议

| 字段 | 建议 |
|---|---|
| `test_name` | 例如 AFP、CA19-9、BI-RADS、PI-RADS、LDCT |
| `test_modality` | serum_marker / imaging / stool_test / molecular_test |
| `cancer_context` | HCC、胰腺癌、乳腺癌、肺癌、CRC、宫颈癌前病变等 |
| `result_or_threshold` | 具体阈值或分级，如 AFP 20-100 ng/mL、BI-RADS 4A、PI-RADS >=4 |
| `evidence_metric` | sensitivity/specificity/AUC/PPV/NPV/malignancy_probability/RR/OR/LR |
| `evidence_value` | 数值保留原始单位和 CI |
| `population` | 高危筛查、疑似病例、术后随访、影像可见病灶等 |
| `source_type` | guideline / systematic_review / meta_analysis / RCT / cohort / review |
| `reference` | PMID/DOI/URL |
| `caution` | 是否不推荐单独筛查、假阳性风险、阈值依赖、人群限制 |
