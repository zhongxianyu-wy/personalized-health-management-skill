# 卵巢癌筛查指南

> 来源：陈小军 等. 卵巢癌早期筛查中国专家共识（2025年版）
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **特点**：卵巢癌起病隐匿，早期症状不典型（腹胀、腹部不适、食欲下降），被称为"沉默的杀手"
- **确诊分期**：约**70%确诊时已是晚期**（III-IV期）
- **5年生存率**：早期>90%，晚期仅30-40%
- **病因**：BRCA1/BRCA2突变是卵巢癌最重要的遗传风险因素

## 高危对象

### 遗传高风险

1. **BRCA1/BRCA2** 基因突变携带者（BRCA1终身风险~40-60%，BRCA2~15-25%）
2. 有卵巢癌/乳腺癌家族史（尤其绝经前发病、双侧乳腺癌）
3. **Lynch综合征**患者（子宫内膜癌+结直肠癌+卵巢癌风险升高）
4. 有其他卵巢癌易感基因突变（RAD51C、RAD51D、BRIP1等）

### 其他风险因素

5. 未产或不孕女性（排卵次数多）
6. 子宫内膜异位症患者（增加透明细胞癌和子宫内膜样癌风险）
7. 激素替代治疗（长期使用）

## 筛查方法

### 联合筛查（推荐方案）

| 方法 | 频率 | 说明 |
|------|------|------|
| 血清 **CA125** | 每6-12个月 | 最常用的卵巢癌标志物（50% I期升高） |
| **阴道超声**（TVS） | 每6-12个月 | 观察卵巢形态、大小、血流 |

### 其他标志物
- **HE4**（人附睾蛋白4）：联用CA125提高准确性
- **ROMA指数**（CA125+HE4+绝经状态）：评估卵巢癌风险
- **ROCA**（卵巢癌风险算法）：动态追踪CA125变化趋势

### 重要提示
- 普通人群**不推荐**卵巢癌筛查（假阳性率高，获益证据不足）
- 筛查仅限于**高危人群**

## 筛查建议

### 高危女性

| 起始年龄 | 方案 |
|----------|------|
| **30-35岁** | 开始每6-12个月 CA125 + TVS |

### BRCA突变携带者

| 措施 | 时机 |
|------|------|
| 开始筛查 | **30岁**起 |
| CA125 + TVS | 每6-12个月 |
| 预防性卵巢输卵管切除术（RRSO） | **35-40岁**（BRCA1）/ **40-45岁**（BRCA2），完成生育后 |
| RRSO降低卵巢癌风险 | **80-96%** |

### 预防性手术
- **RRSO**（双侧输卵管卵巢切除术）是降低遗传性卵巢癌风险最有效的方法
- 同时还降低乳腺癌风险（绝经前切除）

## 预防建议

1. 口服避孕药：长期使用（>=5年）可降低卵巢癌风险约**50%**（但增加乳腺癌和血栓风险，需个体化评估）
2. 母乳喂养（每哺乳12个月降低风险约8%）
3. 输卵管结扎：可降低风险30-40%
4. BRCA突变者完成生育后考虑**预防性手术**
5. 高危人群加强监测

## 关键数据速查

| 指标 | 数据 |
|------|------|
| 晚期确诊比例 | ~70% |
| BRCA1终身卵巢癌风险 | 40-60% |
| BRCA2终身卵巢癌风险 | 15-25% |
| RRSO降低卵巢癌风险 | 80-96% |
| 口服避孕药降低风险 | ~50%（>=5年使用） |
| 普通人群是否筛查 | 不推荐 |
| 高危起始筛查年龄 | 30岁 |

## 新兴筛查技术

卵巢癌目前**缺乏标准无创筛查方法**，CA125+TVS仅推荐用于高危人群。

> **MCED血液检测**：[[../吉早安/癌种覆盖|吉早安]] 对卵巢癌的敏感性为 **87.9%**（Stage I: 50.0%, Stage II: 100%），为卵巢癌无创筛查提供了新选择。
