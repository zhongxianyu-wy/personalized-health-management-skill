# 胰腺癌筛查指南

> 来源：中国胰腺癌高危人群早期筛查和监测共识意见（2021，南京）
> + 上海市抗癌协会 2025版居民恶性肿瘤筛查推荐

## 流行病学

- **趋势**：中国胰腺癌发病率从8/10万上升至**13.52/10万**（呈明显上升趋势）
- **预后极差**：5年生存率<10%，被称为"癌中之王"
- **特点**：起病隐匿，多数确诊时已为晚期；手术切除率仅15-20%

> 胰腺癌早期诊断率极低，筛查仅限于高危人群，不推荐对无症状普通人群进行筛查

## 高危对象

**40岁以上**（特别是50岁以上），符合以下 **>=1种** 情况者：

### 强烈推荐筛查
1. 有胰腺癌**家族史**（一级亲属，尤其>=2位或发病<50岁）
2. 携带胰腺癌易感基因突变（**BRCA2、ATM、PALB2、CDKN2A、STK11**等）
3. 有**糖尿病史**（尤其新近突发糖尿病50岁以上、无家族史）
4. 慢性胰腺炎反复发作（尤其合并胰管结石）
5. 患有主胰管型IPMN、黏液性囊性腺瘤、实性假乳头状瘤

### 考虑筛查
6. 长期吸烟、饮酒、高脂肪高蛋白饮食
7. 有P-J综合征（Peutz-Jeghers综合征）
8. 有Lynch综合征、遗传性胰腺炎等家族史

> **重要**：不推荐对无症状的非高危人群进行胰腺癌筛查

## 筛查方法

### 基础筛查

| 方法 | 灵敏度 | 说明 |
|------|--------|------|
| **CA19-9** + CA125 + CEA | ~80%（CA19-9单独）| 基础肿瘤标志物组合 |
| **腹部B超** | 有限（胰腺深在） | 初筛方法，对小胰癌灵敏度低 |

### 精确检查

| 方法 | 说明 |
|------|------|
| **增强CT**（胰腺薄层扫描）| 筛查首选影像学方法 |
| **MRI/MRCP**（磁共振胆胰管成像）| IPMN等囊性病变随访首选 |
| **超声内镜（EUS）** | 小胰癌（<2cm）灵敏度最高，可见细微病变 |
| **PET-CT** | 鉴别良恶性、评估全身情况 |

### 筛查策略

| 人群 | 方案 | 频率 |
|------|------|------|
| 有家族史/基因突变 | CT或MRI/EUS | 每年1次 |
| 有胰腺囊性病变 | MRI/MRCP | 根据病变大小和特征决定（每年或每半年） |
| 新发糖尿病（>50岁）| CA19-9 + 腹部B超 | 每年1次，密切观察 |

## 筛查流程

```
高危人群
    → CA19-9 + CA125 + CEA + 腹部B超
    → 正常 → 每年1次复查
    → 任一异常 → 增强CT/MRI
                  → 正常 → 3-6个月复查EUS
                  → 可疑 → EUS +/- 穿刺 → MDT
```

## 预防建议

1. **戒烟、控酒**（吸烟增加胰腺癌风险2-3倍）
2. 健康饮食：清淡、易消化、低脂肪，多食禽类鱼虾
3. 控制体重，适度运动
4. 有胰腺癌家族史者**定期体检**（高危人群筛查）
5. 新发糖尿病者应警惕胰腺癌可能（尤其50岁以上、无家族史、体重下降）

## 关键数据速查

| 指标 | 数据 |
|------|------|
| 5年生存率 | <10% |
| 手术切除率 | 15-20% |
| CA19-9灵敏度 | ~80% |
| 筛查范围 | 仅限高危人群 |
| 发病率趋势 | 8→13.52/10万（上升中）|

## 新兴筛查技术

胰腺癌目前**缺乏标准无创筛查方法**，CA19-9灵敏度和特异度均有限。

> **MCED血液检测**：[[../吉早安/癌种覆盖|吉早安]] 对胰腺癌的敏感性为 **66.7%**（Stage I: 60.0%），为胰腺癌无创筛查提供了新选择。
